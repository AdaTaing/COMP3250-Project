#### Dashboard User Guide <br>

**Introduction** <br>
This user guide explains how to use the dashboard created for this project. <br>

**How to Open the Dashboard** <br>
- open Strategy_Analysis_Dashboard.pbix
- verify data source access
- refresh if needed

**Dashboard Features** <br>

- Strategy Comparison
- Portfolio Comparison
- Portfolio value and earnings over time
- KPI Summary Cards
- Risk Metric Comparison
- Filters and Slicers

**How to Use**<br>

- select filters or slicers
- review charts and summary cards
- compare results across categories or time periods
- interpret final outputs

**Troubleshooting**<br>

- refresh the dashboard
- check file paths
- verify required files are present
